Timepstamp Microservice

What's it?

  • API which serves your GET request and returns JSON object like {"unix": 1450137600, "natural": "December 15, 2015"}
  • It can define whether request is natural time or unixtime.

Example requests:

  • https://timestamp-ms.herokuapp.com/1450137600
  • https://timestamp-ms.herokuapp.com/December%2015,%202015
