Timepstamp Microservice

What's it?

  • API which serves GET request and returns JSON object like {"unix": 1450137600, "natural": "December 15, 2015"}
  • It can define whether request is natural time or unixtime.

Example requests:

  • https://timestamp-ms-fc.glitch.me/1450137600
  • https://timestamp-ms-fc.glitch.me/December%2015,%202015
